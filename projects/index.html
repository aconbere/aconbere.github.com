Igor

Get igor at GitHub

Igor is a static blog generator written in python. It writes the blog at anders.conbere.org. The goal is to do the right thing by default, and to use the information provided to us by the surounding tools. If you're using git, we'll use that information, if the file system can provide us details we'll use it. Less reliance on custom tools, more reliance on a structured set of pre-existing tools.

Snake

Get Snake at GitHub

Snake is a clone of the classic game "nibbles (or snake)" into haskell. It uses OpenGL to render the game, using HOpenGL as its surounding framework. It was a chance for me to explore both haskell developement as well as using openGL for more than something trivial.

Announcing Babik

Published on: 2007-03-30 by Anders Conbere

We use MPD here in my office to play media off of my machine. For some time we used a very nice client NeoMPC, however I recently moved machines and sticking with my goal of no longer writing any PHP I decided that it was time to change clients. However I wasn't finding any nice Python or Ruby based web clients, so I wrote my own!

You can find Babik on google code. There aren't any packaged releases, but feel free to grab the source.

It's all written in python, using the lovely py-libmpdclient2 as a back end along with django. To install it on an existing django installation just drop in the source, add babik to your installed apps, and link the media to a static media server.
